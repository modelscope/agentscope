class agentscope.configs.model_config.PostApiCfg[source]

Bases: CfgBase

The config for Post API. The final request post will be

cfg = PostApiCfg(...)
request.post(
    url=cfg.api_url,
    headers=cfg.headers,
    json={
        cfg.message_key: messages,
        **cfg.json_args
    },
    **cfg.post_args
)
api_url: str

The url of the model.

budget: float = None

The total budget using this model. Set to None means no limit.

headers: dict = {}

The headers used for the request.

json_args: dict = None

The additional arguments used within “json” keyword argument, e.g. request.post(json={…, **json_args}), which depends on the specific requirements of the model API.

max_length: int = 2048

The max length of the model, if not provided, defaults to 2048 in model wrapper.

max_retries: int = 3

The max retries of the request.

messages_key: str = 'inputs'

The key of the prompt messages in requests.post(), e.g. request.post(json={${messages_key}: messages, **json_args}). For huggingface and modelscope inference API, the key is inputs

name: str

The name of the model.

post_args: dict = None

The keywords args used within requests.post(), e.g. request.post(…, **generate_args), which depends on the specific requirements of the model API.

timeout: int = 30

The timeout of the request.

type: str = 'post_api'

The typing of this config.
