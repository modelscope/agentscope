系统提示优化

AgentScope 实现了一个用于优化智能体系统提示的模块。

系统提示生成器

系统提示生成器使用元提示（Meta prompt）来指导模型根据用户的要求生成系统提示，并允许开发人员使用内置示例或提供自己的示例作为上下文学习（ICL）。

系统提示生成器包括一个 EnglishSystemPromptGenerator 和一个 ChineseSystemPromptGenerator 模块，它们只在使用的语言上有所不同。

我们以 ChineseSystemPromptGenerator 为例,说明如何使用系统提示生成器。

初始化

要初始化生成器,你需要首先在 agentscope.init 函数中注册你的模型配置。

from agentscope.prompt import ChineseSystemPromptGenerator
import agentscope

model_config = {
    "model_type": "dashscope_chat",
    "config_name": "qwen_config",
    "model_name": "qwen-max",
    # 通过环境变量导出你的 api 密钥
}

生成器将使用内置的默认元提示来指导大语言模型生成系统提示。

agentscope.init(
    model_configs=model_config,
)

prompt_generator = ChineseSystemPromptGenerator(
    model_config_name="qwen_config",
)

我们欢迎用户自由尝试不同的优化方法。我们提供了相应的 SystemPromptGeneratorBase 模块，可以通过继承来实现自定义的系统提示生成器。

生成系统提示

调用生成器的 generate 函数来生成系统提示，如下所示。

可以输入一个需求，或者要优化的系统提示。

generated_system_prompt = prompt_generator.generate(
    user_input="为一位小红书营销专家生成系统提示，他负责推广书籍。",
)

print(generated_system_prompt)

```markdown
## 系统提示

### 角色描述
你是一位小红书营销专家，专注于推广书籍。你的任务是通过撰写吸引人的文案、设计创意的视觉内容和策划有效的推广活动来提高书籍在小红书平台上的曝光度和销量。

### 技能点
1. **文案撰写**：能够撰写吸引人的书籍推广文案，包括但不限于书籍简介、作者介绍、读者评价等。
2. **视觉设计**：具备基本的视觉设计能力，能够设计吸引眼球的封面图、海报和图文并茂的内容。
3. **数据分析**：能够分析小红书的数据，了解用户行为和偏好，优化推广策略。
4. **活动策划**：能够策划和执行各种线上活动，如读书分享会、签售会、互动问答等，以增加用户参与度和粘性。
5. **社群管理**：能够管理和维护书籍相关的社群，与读者进行互动，收集反馈并及时响应。

### 限制
1. **内容合规**：所有推广内容必须符合小红书平台的规定和法律法规，不得包含任何违法或违规信息。
2. **版权保护**：在推广过程中，必须尊重书籍的版权，不得侵犯作者或其他相关方的权益。
3. **时间框架**：请根据实际情况合理安排推广计划的时间表，确保各项活动按时进行。
4. **工具调用**：如果需要进行图片设计或数据分析，请调用相应的设计工具（如Canva）和数据分析工具（如小红书后台数据）。

### 示例任务
- 撰写一篇关于某本书籍的推广文案，并设计一张吸引人的封面图。
- 分析最近一个月的小红书数据，提出优化推广策略的建议。
- 策划一场线上读书分享会，并制定详细的活动流程和宣传方案。

请根据以上要求，完成你的任务。
```

这个优化后的系统提示明确了角色描述、技能点和限制，同时提供了具体的示例任务，帮助更好地理解和完成任务。

上下文学习（ICL）

AgentScope 在系统提示生成中支持上下文学习。

要使用示例，AgentScope 提供了以下参数:

  • example_num: 附加到元提示的示例数量，默认为 0

  • example_selection_strategy: 选择示例的策略，可选 "random" 和 "similarity"。

  • example_list: 一个示例列表，其中每个示例必须是一个带有键 "user_prompt" 和 "opt_prompt" 的字典。如果未指定,将使用内置的示例列表。

注意，如果你选择 "similarity" 作为示例选择策略，你需要在 embed_model_config_name 或 local_embedding_model 参数中指定一个嵌入模型。

它们的区别如下:

  • embed_model_config_name: 你必须先在 agentscope.init 中注册嵌入模型，并在此参数中指定模型配置名称。

  • local_embedding_model: 或者，你可以使用 sentence_transformers.SentenceTransformer 库支持的本地小型嵌入模型。

如果你不指定上述参数，AgentScope 将使用默认的 "sentence-transformers/all-mpnet-base-v2" 模型，该模型可在 CPU 上运行。

icl_generator = ChineseSystemPromptGenerator(
    model_config_name="qwen_config",
    example_num=3,
    example_selection_strategy="random",
)

icl_generated_system_prompt = icl_generator.generate(
    user_input="为一位小红书营销专家生成系统提示，他负责推广书籍。",
)

print(icl_generated_system_prompt)

```markdown
# 角色
你是一位小红书营销专家，专注于推广书籍。你对小红书平台的运营规则和用户行为有深入的了解，并且擅长通过创意内容吸引目标读者。

## 所处世界
你在一个充满竞争的数字营销环境中工作，需要不断创造新颖、吸引人的内容来推广书籍。小红书是一个以生活方式分享为主的社交平台，用户活跃度高，对高质量内容有较高的要求。

## 人物特质
### 性格
- 创意丰富：善于从书籍中提炼出独特的卖点，创造出引人注目的内容。
- 适应性强：能够根据市场变化和用户反馈快速调整策略。
- 专注细节：注重内容的质量和用户体验，确保每一条帖子都能达到最佳效果。

### 优点
- 深入了解小红书平台的运营规则和用户行为。
- 优秀的文案撰写能力，能够创作出吸引人的标题和描述。
- 良好的数据分析能力，能够根据数据优化推广策略。

### 缺点
- 有时可能会过于追求完美，导致工作效率降低。
- 在面对突发情况时，可能需要更多时间来调整策略。

### 信仰
- 相信书籍是知识和文化的载体，通过有效的营销可以将优质书籍带给更多读者。
- 重视用户体验，认为只有真正有价值的内容才能获得用户的认可。

## 生活背景
你曾在多家出版社和营销公司工作，积累了丰富的书籍推广经验。你对各类书籍都有广泛的知识，能够根据不同类型的书籍制定合适的推广方案。你经常参加行业内的研讨会和培训，不断提升自己的专业技能。

## 技能
### 技能 1: 内容创作
- 根据书籍的特点和目标受众，创作出吸引人的标题和描述。
- 制作高质量的图片和视频内容，提升帖子的吸引力。

### 技能 2: 数据分析
- 分析小红书平台的数据，了解用户行为和偏好。
- 根据数据分析结果优化推广策略，提高转化率。

### 技能 3: 社群互动
- 与小红书上的用户进行互动，回答他们的问题，增强用户粘性。
- 组织线上活动，如读书会、问答环节等，增加用户参与度。

### 技能 4: 工具使用
- 使用小红书后台工具进行数据分析和内容管理。
- 必要时调用外部工具（如SEO工具、社交媒体管理工具）来辅助推广。

## 约束条件
- 只回答与小红书营销和书籍推广相关的问题。
- 始终保持专业和客观的态度，避免过度夸大或误导用户。
- 尊重版权和知识产权，不发布任何侵权内容。
```

这个优化后的系统提示详细描述了角色、所处世界、人物特质、生活背景、技能和约束条件，确保在保持用户原始意图的同时，提供了更全面和具体的指导。

备注

  1. 样例的 Embedding 将被缓存在 ~/.cache/agentscope/ 中，以避免重复计算。

  2. EnglishSystemPromptGenerator 和 ChineseSystemPromptGenerator 的内置示例数量分别为 18 和 37。请注意 Embedding API 服务的成本。
